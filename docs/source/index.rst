Flask-PyMODM
================
Flask-PyMODM is the bridge between `Flask`_ and `PyMODM`_, it brings to developers a simplified way to use PyMODM in Flask applications.

.. _PyMODM: https://pymodm.readthedocs.io/en/stable/
.. _Flask: http://flask.pocoo.org/

See `the PyMODM documentation`_ to learn how to work with the ODM in depth.

.. _the PyMODM documentation: https://pymodm.readthedocs.io/en/stable/
